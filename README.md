# Вита-Грин

Коммерческий лендинг вымышленного бренда жидких удобрений и микроэлементных комплексов для питания растений.

Проект показывает, как можно упаковать нишевый агро-продукт в понятный русскоязычный сайт: первый экран, задачи, продуктовая линейка, культуры, аудитории, подбор решения, партнерский блок и форма заявки.

## Стек

- Next.js
- React
- Tailwind CSS
- статический экспорт
- форма без серверной отправки

## Запуск локально

```bash
npm install
npm run dev
```

Открыть локально:

```text
http://localhost:3000
```

## Сборка

```bash
npm run build
```

После сборки Next.js создает статический экспорт в папке `out`. Ее можно загрузить на обычный shared-хостинг через файловый менеджер, FTP или SFTP.

## Сборка для GitHub Pages

Для публикации на `https://rngoblin.github.io/agro-dev/` проект собирается с базовым путём `/agro-dev`:

```bash
GITHUB_PAGES=true npm run build
```

## Проверка качества

```bash
npm run lint
```
