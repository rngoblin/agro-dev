# Концепт лендинга для B2B-направления жидких микроудобрений

Рабочее название: **Демо-кейс B2B-лендинга для жидких микроудобрений**.

Это обезличенный демонстрационный лендинг для портфолио веб-дизайнера/фронтенд-разработчика. Проект показывает структуру, UX-логику, визуальную систему и адаптивную фронтенд-сборку B2B-лендинга для производственного продукта.

## Важное ограничение

Проект не является официальным сайтом какой-либо компании и не содержит:

- реальные финансовые показатели;
- реальные имена клиентов;
- реальные договорные условия;
- внутренние документы;
- реальные прайс-листы;
- производственные мощности;
- коммерческие обещания, которые невозможно подтвердить.

## Структура страницы

1. Первый экран.
2. Проблема.
3. Решение.
4. Продуктовая линейка.
5. Клиентские сценарии.
6. Процесс.
7. До / после.
8. Короткий блок роли и результата.
9. Форма заявки.
10. Футер с демо-статусом.

## Роль исполнителя

Структура, UX-логика, веб-дизайн, фронтенд-сборка, адаптив.

## Демонстрационные данные

- Название проекта условное.
- Карточки продуктов условные.
- Форма не отправляет данные.
- Изображение используется как условный визуальный пример.
- Страница `/case` описывает работу как кейс для портфолио.
