# Демонстрационные данные

Этот проект подготовлен как публичный демо-кейс для портфолио. Все данные обезличены и не должны восприниматься как реальные коммерческие материалы.

## Где используются демонстрационные данные

- Название проекта — условное название интерфейсного концепта.
- Первый экран — демонстрационный текст, описывающий категорию продукта без привязки к компании.
- Продуктовая линейка — условные карточки продуктов: комплекс микроэлементов, бор, цинк, азот, фосфор, калий, молибден, гуматы.
- Описания продуктов — обобщённые тексты без составов, дозировок и норм применения.
- Клиентские сценарии — обобщённые B2B-аудитории.
- Процесс — типовой коммерческий сценарий без реальных договорных условий.
- Блок до / после — типовая логика редизайна, не сравнение с реальным старым сайтом.
- Форма заявки — форма не отправляет данные и показывает демонстрационное сообщение.
- Изображение первого экрана — условный визуальный пример для производственного B2B-кейса.

## Что не используется

- Реальные финансовые показатели.
- Реальные имена клиентов.
- Реальные договорные условия.
- Внутренние документы.
- Реальные прайс-листы.
- Реальные производственные мощности.
- Неподтверждённые коммерческие обещания.
- Формулировки “официальный сайт компании”.

## Публичная публикация

Лендинг можно публиковать как демо-кейс при сохранении дисклеймера:

```text
Демо-кейс для портфолио. Не является официальным сайтом компании.
```
