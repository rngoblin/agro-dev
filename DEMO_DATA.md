# Условные данные проекта

Проект подготовлен как концепт лендинга для вымышленного бренда «Вита-Грин».

## Где используются условные данные

- Название бренда «Вита-Грин».
- Продуктовая линейка: комплекс, бор, цинк, молибден, калий, фосфор, азот, гумат.
- Описания продуктов без составов, дозировок и норм применения.
- Аудитории и культуры как типовые сценарии для агро-лендинга.
- Форма заявки без серверной отправки.
- Визуальные мокапы упаковки и карточек, созданные средствами интерфейса.

## Что не используется

- Реальные финансовые показатели.
- Реальные клиенты.
- Реальные прайс-листы.
- Реальные договорные условия.
- Внутренние документы.
- Производственные данные.
- Гарантированные обещания результата.
