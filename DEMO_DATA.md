# Demo / placeholder data

Этот проект подготовлен как публичный portfolio demo case. Все данные обезличены и не должны восприниматься как реальные коммерческие материалы.

## Где используются demo-данные

- Название `Liquid Concept` — условное название интерфейсного концепта.
- Hero copy — демонстрационный текст, описывающий категорию продукта без привязки к компании.
- Product line section — условные карточки продуктов: комплекс микроэлементов, бор, цинк, азот, фосфор, калий, молибден, гуматы.
- Product descriptions — нейтральные placeholder-тексты без составов, дозировок и норм применения.
- Audience section — обобщённые B2B-аудитории.
- Process section — типовой коммерческий сценарий, без реальных договорных условий.
- Before / After section — типовая логика редизайна, не сравнение с реальным старым сайтом.
- Demo contact form — форма не отправляет данные и показывает сообщение `Demo form`.
- Hero image — нейтральный placeholder-визуал для атмосферы производственного B2B-кейса.

## Что не используется

- Реальные финансовые показатели.
- Реальные имена клиентов.
- Реальные договорные условия.
- Внутренние документы.
- Реальные прайс-листы.
- Реальные производственные мощности.
- Неподтверждённые коммерческие обещания.
- Формулировки “официальный сайт компании”.

## Публичная публикация

Лендинг можно публиковать как portfolio demo при сохранении disclaimer:

```text
Demo portfolio case. Not an official company website.
```
