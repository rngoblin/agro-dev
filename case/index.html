Портфолио-кейс

Вита-Грин — концепт лендинга для бренда жидких удобрений

Вита-Грин — вымышленный бренд, созданный как демонстрационный проект для портфолио. Задача кейса — показать, как нишевый агро-продукт можно упаковать в понятный лендинг: с продуктовой линейкой, разными аудиториями, консультационным сценарием и формой заявки.

Портфолио

Как устроен кейс

Эта страница честно описывает концепцию, структуру и роль в проекте. Сам лендинг при этом работает как самостоятельный брендовый сайт.

Задача

Создать самостоятельный лендинг для вымышленного бренда жидких удобрений, который выглядит как реальный коммерческий сайт и демонстрирует подход к структуре, тексту, визуалу и фронтенд-сборке.

Подход

Сайт строится вокруг понятной продуктовой линейки, задач пользователя и сценария подбора решения.

Структура

Первый экран, задачи, продуктовая линейка, аудитории, подбор продукта, преимущества, партнерский блок и форма заявки.

Роль

Концепция, структура, UX-логика, текстовая подача, веб-дизайн, фронтенд-сборка, адаптив.

Стек

Next.js static export, React, Tailwind CSS, статическая сборка.

Что показывает кейс

Кейс показывает способность работать с нишевым B2B/B2C-продуктом, объяснять сложную линейку простым языком и собирать коммерческий лендинг без перегруза.
